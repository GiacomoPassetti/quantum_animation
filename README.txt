The scope of this (ambicious) folder is to implement a visualization for the real space spreading of a single particle wave function.
Final result will be an interactive UI where the user can define the shape of a potential, and the size of the initial particle confinement.
The user will be given a set of default potentials, like harmonic oscillator and periodic wells/waves. But it will also be possible to set custom potentials.
The wavefunction at t=0 will be set with a uniform distribution within a circle centered at the origin and with radius given by the confinement setting. 
Then with play real time simulation starts showing the time evolution of the wave function. For visual there will be a black background, while the wavefunction will be represented
by a soft orange/red colormap with both transparecy and a glowing effect. Colors and effect can be changed in metasettings.
